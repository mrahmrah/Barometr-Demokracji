# ⚖️ Barometr Demokracji
Projekt analizujący standardy demokratyczne w tekstach przy użyciu Google Gemini AI.

## 🚀 Funkcje
- Analiza tekstów pod kątem populizmu i manipulacji.
- Wykorzystanie modelu Gemini 1.5 Flash.
- Interfejs zbudowany w czystym JavaScript (Vite).

## 🛠️ Jak uruchomić lokalnie?
1. Sklonuj repozytorium.
2. W folderze `ui` wykonaj `npm install`.
3. Stwórz plik `.env` i dodaj swój klucz: `VITE_GEMINI_API_KEY=TWÓJ_KLUCZ`.
4. Uruchom przez `npm run dev`.
